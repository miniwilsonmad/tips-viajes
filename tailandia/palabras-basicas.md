# Palabras básicas en tailandés 🇹🇭

Fuente: [@jum_thitapa — Instagram Reel](https://www.instagram.com/reel/DW1NEbWD8P_/)

6 palabras esenciales para el día a día.

---

## Las 6 palabras

| # | Tailandés | Pronunciación | Significado |
|---|---|---|---|
| 1 | **ใช่** | *Châi* | Sí |
| 2 | **ไม่ใช่** | *Mâi châi* | No |
| 3 | **ดี** | *Dii* | Bueno / Bien |
| 4 | **ไม่ดี** | *Mâi dii* | No bueno / Mal |
| 5 | **โอเค** | *O-kee* | OK / Está bien |
| 6 | **ไม่โอเค** | *Mâi o-kee* | No OK / No está bien |

---

## Notas

- **"Mâi"** (ไม่) es el prefijo universal para negar: anteponlo a cualquier palabra para decir lo contrario.
  - ใช่ (châi) = sí → ไม่ใช่ = no
  - ดี (dii) = bueno → ไม่ดี = malo
  - โอเค (o-kee) = ok → ไม่โอเค = no ok
- **"Dii" (ดี)** es súper versátil: significa bueno, bien, sabroso, bonito... ¡todo lo positivo!
- **"O-kee" (โอเค)** es un préstamo del inglés, se usa exactamente igual que en español.
