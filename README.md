# Tips Viajes 🌏

Repositorio de tips, recomendaciones y recursos extra para cada país de La Vuelta al Mundo.
No es el plan maestro (ese está en `Viajes/`). Esto es la mochila digital: lo que encontramos en reels, videos, blogs, etc., **siempre verificado contra fuentes oficiales**.

## Estructura

```
tips-viajes/
├── README.md                  ← este archivo
├── vietnam/
│   ├── README.md              ← índice de tips Vietnam
│   └── tren-nocturno-lujo.md  ← ejemplo: tren VIP sleeper
├── japon/
│   └── ...
├── tailandia/
│   └── ...
└── ...
```

## Metodología

1. **El reel/video es el punto de partida**, no la fuente definitiva
2. **Bucear fuentes oficiales**: sitios web de las compañías, portales gubernamentales, guías oficiales
3. **Verificar precios, rutas, disponibilidad** contra múltiples fuentes
4. **Guardar aquí la versión completa y verificada**
5. **Citar siempre las fuentes** (enlace al reel original + fuentes oficiales consultadas)

## Cómo usar

Cuando Pablo envíe un reel diciendo "guarda esto para [país]":
1. Analizar el reel (AIR pipeline)
2. Investigar fuentes oficiales con web_search
3. Crear/actualizar el archivo `.md` en la carpeta del país
4. Responder con el resumen de lo guardado
